# شبیه‌ساز جام جهانی فوتبال (World Cup Simulator)

یک شبیه‌ساز کامل از یک دوره جام جهانی فوتبال با زبان پایتون. برنامه اطلاعات
۳۲ تیم را از یک فایل CSV می‌خواند، گروه‌ها را قرعه‌کشی و سیدبندی می‌کند،
مرحله گروهی و مرحله حذفی را شبیه‌سازی می‌کند و در نهایت قهرمان تورنمنت را
مشخص می‌کند. همچنین امکان اجرای هزاران‌باره کل تورنمنت برای محاسبه درصد
شانس قهرمانی هر تیم و رسم نمودار نتایج نیز وجود دارد.

## امکانات

- بارگذاری اطلاعات تیم‌ها از یک فایل CSV
- قرعه‌کشی و سیدبندی خودکار ۸ گروه بر اساس رنک فیفا (۴ سید × ۸ تیم)
- شبیه‌سازی نتیجه گل هر بازی با توزیع آماری پواسون
- اجرای کامل مرحله گروهی و رتبه‌بندی تیم‌ها (امتیاز، تفاضل گل، گل زده، نتیجه رودررو)
- اجرای مرحله حذفی (یک‌هشتم نهایی تا فینال) با وقت اضافه و ضربات پنالتی در صورت تساوی
- اجرای تکراری تورنمنت (پیش‌فرض ۱۰۰۰ بار) برای محاسبه درصد شانس قهرمانی هر تیم
- رسم نمودار میله‌ای درصد شانس قهرمانی تیم‌های برتر (اختیاری، با matplotlib)
- رابط کاربری متنی (منو) برای اجرای مرحله‌به‌مرحله یا کامل برنامه

## فایل ورودی (CSV)

برنامه انتظار یک فایل CSV با نام پیش‌فرض `worldcup_2026_teams.csv` (یا هر
مسیر دیگری که وارد کنید) و دقیقاً ۴ ستون زیر را دارد:

| ستون | توضیح |
|---|---|
| `name` | نام تیم |
| `attack` | امتیاز خط حمله (۰ تا ۱۰۰) |
| `defense` | امتیاز خط دفاع (۰ تا ۱۰۰) |
| `rank` | رنک فیفا (برای سیدبندی گروه‌ها استفاده می‌شود) |

نمونه:

```csv
name,attack,defense,rank
Argentina,90,85,1
France,88,84,2
Brazil,87,83,3
...
```

فایل باید شامل دقیقاً **۳۲ تیم** باشد تا ساختار ۸ گروه ۴ تیمی به‌درستی ساخته شود.

## نحوه اجرا

```bash
python worldcup_simulator.py
```

پس از اجرا، منوی زیر نمایش داده می‌شود:

```
===== Shabih saze Jame Jahani =====
1) Bargozarie team ha az file CSV
2) Anjame ghorekeshie group ha (seed bandi e khodkar)
3) Ejraye marhaleye goroohi va namayeshe jadvale har group
4) Ejraye kamele Jam (goroohi + hazfi) va namayeshe ghahreman
5) Shabihsazie 1000 bare va gozareshe darsade ghahremani
6) Namayeshe brackete hazfie akharin shabihsazi
7) Namayeshe nemoodare ehtemale ghahremani
8) Khorooj
```

| گزینه | عملکرد |
|---|---|
| 1 | بارگذاری تیم‌ها از فایل CSV |
| 2 | قرعه‌کشی و سیدبندی گروه‌ها |
| 3 | اجرای مرحله گروهی و نمایش جدول رده‌بندی هر گروه |
| 4 | اجرای کامل یک دوره جام جهانی (گروهی + حذفی) و نمایش قهرمان |
| 5 | اجرای شبیه‌سازی چندباره (پیش‌فرض ۱۰۰۰ بار) و گزارش درصد شانس قهرمانی هر تیم |
| 6 | نمایش براکت حذفی مربوط به آخرین اجرا |
| 7 | رسم نمودار میله‌ای درصد شانس قهرمانی تیم‌های برتر (نیازمند اجرای قبلی گزینه ۵) |
| 8 | خروج از برنامه |

برای شروع سریع، معمولاً کافی است گزینه‌های **۱** و سپس **۴** را انتخاب کنید
تا یک دوره کامل جام جهانی اجرا و قهرمان مشخص شود.

## ساختار کلاس‌ها

| کلاس | مسئولیت |
|---|---|
| `Team` | نگهداری اطلاعات و آمار یک تیم؛ شبیه‌سازی گل هر بازی و ضربات پنالتی |
| `Match` | اجرای یک مسابقه (وقت اصلی، وقت اضافه و پنالتی در مرحله حذفی) |
| `Group` | نگهداری تیم‌های یک گروه، اجرای بازی‌های داخل گروه و رتبه‌بندی آن‌ها |
| `KnockoutStage` | مدیریت یک دور از مرحله حذفی (ساخت، اجرا و استخراج برندگان) |
| `WorldCupSimulator` | هماهنگ‌کننده اصلی؛ بارگذاری داده، قرعه‌کشی، اجرای کل تورنمنت و گزارش‌گیری آماری |

## نکات فنی

- تعداد گل هر بازی از توزیع پواسون نمونه‌برداری می‌شود؛ میانگین آن ترکیبی
  از قدرت حمله تیم خودی و قدرت دفاع تیم حریف است.
- در مرحله گروهی، ترتیب صعود بر اساس امتیاز، تفاضل گل، گل زده، نتیجه بازی
  مستقیم و در نهایت قرعه‌کشی تصادفی تعیین می‌شود.

  ۰.۳۳) و در صورت نیاز ضربات پنالتی شبیه‌سازی می‌شود.
